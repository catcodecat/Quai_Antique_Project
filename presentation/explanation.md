# План презентации Quai Antique

## 1. Общая идея проекта

Сайт называется Quai Antique. Это витрина ресторана с главной страницей,
галереей, картой блюд, бронированием и страницей входа. Я придерживалась
цветовой гаммы из макета: коричневый фон, золотистые кнопки, светлые секции.

Показать:

- файл `index.html`;
- файл `styles.css`;
- файл `app.js`;
- страницу сайта в браузере.

## 2. Header

В `index.html` header начинается с тега `<header class="site-header">`.
Внутри есть название ресторана `Quai Antique` и навигация `<nav>`.

Что объяснить:

- `header` отвечает за верхнюю часть сайта;
- `nav` хранит ссылки на страницы;
- ссылки ведут на `index.html`, `gallery.html`, `menu.html`,
  `reservation.html`, `login.html`.

В `styles.css` показать:

- `.site-header` - фон, расположение элементов, отступы;
- `.brand` - стиль названия ресторана;
- `.main-nav` - стиль меню.

Скриншот:

- верхняя часть главной страницы с меню.

## 3. Hero-блок

Hero-блок находится в `index.html`:

```html
<section class="hero hero-home">
```

Что объяснить:

- это первый большой визуальный блок страницы;
- в нём есть текст, название ресторана и кнопка `Réserver`;
- фон добавлен через CSS с помощью `background`.

В `styles.css` показать:

- `.hero`;
- `.hero h1`;
- `.btn`.

Скриншот:

- первый экран сайта с названием Quai Antique.

## 4. Body / основное содержимое

Основная часть сайта находится внутри `<main>`.

На главной странице есть блоки:

- `Bienvenue au quai Antique`;
- `Des produits frais`;
- `Réservation en Ligne`.

Что объяснить:

- каждый блок сделан через `<section>`;
- для текста и изображения используется сетка CSS Grid;
- светлые и тёмные блоки чередуются, как в примере.

В `styles.css` показать:

- `.section-light`;
- `.section-dark`;
- `.intro-grid`;
- `.split`.

Скриншот:

- блок приветствия с текстом и фото шефа;
- блок `Des produits frais`.

## 5. Galerie

Страница `gallery.html` показывает изображения ресторана.

Что объяснить:

- карточки сделаны через `<article class="gallery-card">`;
- сетка изображений сделана через `.gallery-grid`;
- изображения лежат в папке `assets/images`.
- для лучшего качества используются фотографии с большим разрешением,
  сохранённые локально в папке `assets/images`.

Скриншот:

- страница галереи.

## 6. La carte / меню

Страница `menu.html` показывает категории блюд.

Что объяснить:

- категории меню сделаны через `<article class="menu-category">`;
- каждое блюдо находится в блоке `.dish`;
- цена выделена тегом `<strong>`.
- вкладки `Entrées`, `Plats`, `Desserts` переключаются с помощью JavaScript
  в файле `app.js`.

В `styles.css` показать:

- `.tabs`;
- `.menu-list`;
- `.menu-category`;
- `.dish`.

Скриншот:

- страница `La carte`.

## 7. Reservation

Страница `reservation.html` содержит форму бронирования.

Что объяснить:

- форма создана тегом `<form>`;
- поля сделаны через `input`, `select`, `textarea`;
- кнопка использует общий класс `.btn`.

Скриншот:

- форма бронирования.

## 8. Connexion

Страница `login.html` показывает форму входа.

Что объяснить:

- используется `input type="email"` для email;
- используется `input type="password"` для пароля;
- внешний вид формы общий для всех страниц через класс `.form-panel`.
- поля оставлены пустыми;
- ссылка `Inscrivez-vous ici !` ведёт на страницу `register.html`.

Скриншот:

- страница входа.

## 9. Footer

Footer находится внизу каждой страницы:

```html
<footer class="site-footer">
```

Что объяснить:

- footer содержит расписание, адрес и контакт;
- структура повторяется на всех страницах;
- стиль находится в `.site-footer`.

Скриншот:

- нижняя часть страницы.

## 10. Адаптивность

В конце `styles.css` есть блок:

```css
@media (max-width: 760px)
```

Что объяснить:

- на маленьком экране меню и секции становятся в одну колонку;
- изображения и карточки не выходят за границы экрана;
- сайт можно открыть на телефоне.

Скриншот:

- главная страница в узком размере окна.
